
# human_following_robo

Human-following robot with manual control and obstacle avoidance capabilities.






# Components Used:
    -> Arduino Uno
    -> Arduino Motor Driver shield
    -> Jumper wires
    -> 4 BO motors
    -> Hook up wires
    -> 9V battery
    -> 4 wheel
    -> Chassis
    -> Ultrasonic sensor
    -> 2 IR Sensor
    -> on-off switch
    -> servo motor
    -> Bluetooth Module HC-05
